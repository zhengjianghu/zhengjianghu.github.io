Git使用笔记

命令行优先

git stash

用来暂存修改，将其修改把保存到栈上
  • git stash save = git stash
  • git stash list：查看之前被储藏的应用
  • git stash apply XXX：将之前被储藏的工作重新应用起来
  • git stash drop XXX：移除某个储藏
  • git tash pop:应用该储藏并将其立即从栈上扔掉

git reset

版本恢复指令
  • git reset --hard：彻底回到某个版本，本地的源码也会变为上一个版本的内容
  • git reset --soft：回退到某个版本，但是只回退了commit的信息，不会恢复到index file一级。

Git pull错误

error: Your local changes to the following files would be overwritten by merge: diploma project/毕业设计.docx diploma project/遗留问题备注.docx Please commit your changes or stash them before you merge. Aborting Updating d1b37b4..b187270
原因：表明从服务器拉取的文件跟本地文件有重复，提示会否需要进行本地版本的保存。
解决办法：(命令行含义在上面进行讲解)
  • 对本地文件进行版本保存然后再拉取
    • git stash
    git pull
    git stash pop
  • 不保存直接覆盖
    • git reset --hard
    git pull
