嵌入式开发工具汇总 嵌入式软/硬件开发工具合集

硬件开发工具

图纸查看

  • CAM350：查看最终的电路制版gerber文件
  • CAD快速看图：查看板子图纸

绘图

  • OrCAD Capture：绘制原理图

软件开发工具

代码编辑/编译

  • Eclipse：代码编辑/编译器、IDE
  • Visual Studio Code：多功能代码编辑器
  • Sublime text：代码编辑器
  • IAR Embedded Workbench：IDE，MCU内核开发，类似于keil

文本编辑

  • Notepad++：万用文本编辑器
  • UltraEdit

程序烧写/调试

  • Serial Port Utility：串口调试助手
  • 网络调试助手V4.2：网络调试工具
  • Socktool：Socket调试工具
  • JFlash：JLink代码烧写

Linux系统相关

  • VMware-workstation：装载虚拟系统平台软件
  • Orcal VM Vitrual Box：虚拟系统装载平台
  • Xftp 5/Xshell 5：shell命令调试/文件传输
  • putty：Linux系统远程登录工具
  • MobaXterm：好用的Linux远程调试/文件传输软件

前端开发

  • MarkdownPad2：markdown语言编辑器
  • postman：http请求模拟工具
  • JetBrains WebStorm：前端网页开发软件
  • Atom：强大的前端开发软件

后台开发

  • WindowsPowerShell：Windows环境下下shell调试工具

其他

  • Axure RP：产品构建
  • Everything：文件搜索工具
  • BeyondCompare：代码比较器
  • viso：office自带的绘图软件
  • 亿图图示：国产良心绘图软件
